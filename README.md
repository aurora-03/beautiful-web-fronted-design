## Overview

**Area** is a modern, minimalist product landing page for a data analytics platform. The design emphasizes clean typography, generous whitespace, and a sophisticated olive-green color palette — creating a premium feel that lets the content breathe.

> From Figma design to production-ready code, every detail is faithfully translated.

---

## Design Highlights

### Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| Dark Olive | `#485C11` | Primary buttons, captions, accents |
| Mid Green | `#8E9C78` | Hero background |

## Responsive Design

Three breakpoints ensure a seamless experience across all devices:

```
Desktop  →  1200px+   Full layout, 1500px max-width
Tablet   →  900–1200px  Stacked features, adjusted hero
Mobile   →  < 900px    Single column, hidden floating nav
```

---

## Interactions

- **Smooth Scroll** — Anchor links animate to sections
- **Floating Nav** — Auto-hides on scroll down, reappears on scroll up
- **Fade-in Animation** — Sections reveal as they enter the viewport
- **Button Hover States** — Subtle background transitions
- **Glassmorphism** — Frosted glass effect on floating navigation
